# 过期

一直以来 expires 与 last-modified 是让人困惑的点。

  • expires 是设置过期时间，没有过期就不会访问服务器
  • last-modified 是设置上次的更新时间，每次都会向服务器查询最近的更新时间

让人难过的事情莫过于在理解后，才发现别人早已对这部分有很好的概括了open in new window。
